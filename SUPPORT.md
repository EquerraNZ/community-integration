# Support

## How to file issues and get help

This is a community-maintained repository. For bugs, questions, and feature requests, please open a GitHub issue on this repository. Search existing issues first in case your question has already been answered.

For help with the underlying tooling:

- AL-Go for GitHub: [github.com/microsoft/AL-Go](https://github.com/microsoft/AL-Go)
- Business Central developer docs: [aka.ms/bcdocs](https://aka.ms/bcdocs)
- BCQuality rules: [github.com/EquerraNZ/community-BCQuality](https://github.com/EquerraNZ/community-BCQuality)

## Support policy

Support is provided on a best-effort basis by the community maintainers. There is no commercial support or service-level agreement.
